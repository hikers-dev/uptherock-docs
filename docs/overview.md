# Overview

## 서비스 개요

**오르락(UpTheRock)** 은  
등산이라는 오프라인 활동을 디지털 콘텐츠로 확장한  
위치 기반 게이미피케이션 서비스입니다.

사용자의 실제 등산 데이터를 기반으로  
등산 과정을 하나의 **플레이 가능한 경험**으로 재구성하는 것을 목표로 합니다. 

---

## 문제 인식

최근 등산 인구는 지속적으로 증가하고 있으며,  
등산은 하나의 라이프스타일로 자리 잡고 있습니다.  
그러나 등산 문화의 디지털 전환은 충분히 이루어지지 않았다고 판단했습니다.

기존 등산 관련 서비스들은 주로 다음과 같은 한계를 가집니다.

- 기록 중심의 단순한 운동 앱 구조
- 반복 등산 시 경험의 변화 부족
- 사용자 간 관계 형성이 어려운 구조

즉,  
> **등산은 재미있지만, 디지털 경험은 단조롭다**  
는 문제의식에서 오르락은 출발했습니다.

---

## 접근 관점

오르락은 등산을  
단순한 운동 기록이 아닌 **엔터테인먼트 콘텐츠**로 바라봅니다.

이를 위해 다음과 같은 관점을 적용했습니다.

- 실제 산과 실제 경로를 콘텐츠의 중심으로 사용
- 등산 과정을 ‘코스 단위 경험’으로 구조화
- 완주, 반복, 도전이라는 요소를 통해 지속적인 참여 유도

이러한 접근은  
기존 등산 앱의 **기록 열람 중심 구조**에서 벗어나,  
사용자가 등산 자체를 하나의 콘텐츠로 인식하도록 만듭니다.  

---

## 핵심 개념

### 1. 등산 게이미피케이션

- 실제 GPS 이동 경로를 기반으로 등산을 기록합니다.
- 기록된 등산 경로는 ‘코스’로 관리되며, 반복과 축적이 가능합니다.
- 사용자는 등산을 통해 성취와 수집의 경험을 얻게 됩니다.

### 2. 디지털 콘텐츠 기반 버티컬 커뮤니티

- 등산이라는 명확한 주제를 중심으로 한 커뮤니티 구조를 지향합니다.
- 사용자 개개인의 등산 경험이 콘텐츠가 되고,  
  그 콘텐츠가 다시 참여를 유도하는 구조를 만듭니다.  

---

## 오르락이 지향하는 방향

오르락은 단순한 등산 기록 앱이 아닌,

- 등산 경험을 디지털로 확장하고
- 사용자의 행동 변화를 유도하며
- 실제 활동 데이터가 콘텐츠 자산으로 축적되는 서비스

를 지향합니다.

> **세상에 없던 ‘오르는 즐거움’을 만드는 것**  
> 이것이 오르락의 목표입니다.
