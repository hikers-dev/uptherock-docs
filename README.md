# UpTheRock

> 등산 경험을 디지털로 확장한  
> 위치 기반 게이미피케이션 서비스

## ✍️ 서비스 소개

**오르락(UpTheRock)**은 사용자의 실제 등산 활동을 기록하고,  
이를 **게임 요소와 결합해 하나의 ‘경험’으로 확장**하는 등산 게이미피케이션 서비스입니다.
